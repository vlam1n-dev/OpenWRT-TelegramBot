# Что это?

**OpenWRT Telegram Bot** — это легковесный, быстрый и безопасный Telegram Бот для удаленного мониторинга и управления роутерами на базе OpenWRT. Написан полностью на чистом Shell с использованием встроенного API роутера (`ubus`, `uci`, `jsonfilter`) и API Telegram.

Настройка конфигурационного файла доступна как через конфиг-файл `/etc/config/telegram`, так и через **LuCI** веб-интерфейс - настройка токена, списка администраторов, уведомлений в боте, проверка обновлений и вывод логов `Services -> Telegram Bot`

## Архитектура

- **Shell**: Чистый `/bin/sh`;
- **Низкоуровневые утилиты**: Для разбора JSON используется встроенный бинарный парсер `jsonfilter`;
- **Обмен данными**: Сетевые запросы осуществляются через `curl`;
- **Сохранение ресурса Flash-памяти**: Временные файлы располагаются в каталоге `/tmp`;
- **Встроенный демон**: Бот работает как системная служба под управлением супервизора `procd`.

---

## Системные требования

- **Операционная система**: OpenWRT **версии 19.07 и выше**;
- **Базовые зависимости** (устанавливаются автоматически):
  - `curl`;
  - `jsonfilter`;
  - `ca-certificates` / `ca-bundle`.
- **Дополнительные зависимости** (устанавливаются по желанию в `install.sh` для работы расширенных модулей):
  - `etherwake` или `wol` - для функции Wake-on-LAN;
  - `vnstat` / `vnstatd` - для сбора статистики общего трафика;
  - `nlbwmon` - для учёта потребления сети локальными клиентами.

## Основные возможности

- **📊 Мониторинг системы**:
  - Просмотр **аптайма**, **версии ПО**, **свободной памяти** ОЗУ/Flash, **загрузки процессор**а CPU, **WAN IP**-адресов;
  - **Запуск** стандартных **сетевых утилит** с динамическим выводом результатов в реальном времени (Ping, Traceroute, DNS Lookup, Port Check).
- **⚡ Управление сетевыми интерфейсами**:
  - **Включение/выключение** любого **сетевого интерфейса**, доступного в OpenWRT;
  - Просмотр детальной информации о каждом сетевом интерфейсе.
- **📱 Список подключенных устройств**
  - Мгновенный **вывод активных DHCP-аренд** с детальной информацией, включающей IP, интерфейс подключения и многое другое;
  - Возможность отключить от сети любое устройство;
  - Возможность заблокировать устройство для последующего подключения к данному сетевому интерфейсу;
  - Возможность отправить **Wake-on-Lan** пакет.
- **🔀 Проброс портов (NAT)**
  - **Управление файрволом** роутера без входа в LuCI;
  - Создание, изменение, включение/отключение уже существующих правил.
- **🔄 Перезагрузка**
  - Перезагрузка любого сетевого интерфейса, включая сам роутер.
- **🔔 Уведомления**
  - **Настройка уведомлений** через Telegram бот (Подключение устройства/Изменение WAN IP/Высокая нагрузка на систему).
- **🌐 Локализация**
  - **Поддержка 4-х языков**, включает Русский, Английский, Украинский, Немецкий.
- **📁 Интеграция с LuCI**
  - Настройка API ключа, ID администраторов, уведомлений через бота, проверка обновлений и многое другое;
  - Просмотр детальных логов работы Telegram Бота.

## О потреблении

* **Оперативная память**: **менее 2 МБ** во время работы. В режиме ожидания нагрузка практически равна нулю;
* **Постоянная память (Flash)**: **менее 60 КБ** для всей установки;
  - Все изменяемые файлы (база данных состояний пользователей, временные FIFO-каналы, PID-файлы, логи) вынесены в директорию `/tmp`.
* **Нагрузка на CPU**: Менее **1%**.

## Установка

Для установки требуется OpenWRT версии `19.07` и выше.

**ВНМИАНИЕ**: Тестирование проводилось только мной на версии `25.12.0`, на других версиях могут быть ошибки!

1. Подключитесь к роутеру (например, используя [Putty](https://github.com/putty-org-ru/PuTTY/releases));
2. Запустите **команду для автоматической загрузки** и запуска установщика:
   ```bash
   wget -qO- https://raw.githubusercontent.com/vlam1n-dev/OpenWRT-TelegramBot/main/install.sh | sh
   ```
3. Пройдите установщик, следуя указаниям.

После успешной установки перейдите в веб-интерфейс LuCI роутера (если был установлен): `Services -> Telegram Bot`, введите токен вашего бота (полученный у [@BotFather](https://t.me/BotFather)) и добавьте свой **Telegram ID** в список администраторов.

Обновление происходит точно так же, запустив `install.sh` укажите **новую версию** или установите **автоматически**.

## Удаление

Для полного и безопасного удаления бота и всех его компонентов, запустите деинсталлятор:
```bash
wget -qO- https://raw.githubusercontent.com/vlam1n-dev/OpenWRT-TelegramBot/main/uninstall.sh | sh
```

Вы можете выбрать, какие **компоненты удалить**, а какие **оставить**, в том числе - **конфигурационный файл**.
