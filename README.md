# 🤖 OpenWRT Telegram Bot

**OpenWRT Telegram Bot** — это ультра-легковесный, быстрый и безопасный Telegram-бот для удаленного мониторинга и управления роутерами на базе OpenWRT. Написан полностью на чистом командном интерпретаторе POSIX Shell с использованием встроенного API роутера (`ubus`, `jsonfilter`), что делает его идеальным решением для встраиваемых систем с жестким ограничением ресурсов.

Проект включает в себя полноценный веб-интерфейс администратора для **LuCI**, позволяющий настраивать токен, список администраторов, уведомления и логи прямо через стандартный браузерный кабинет роутера.

---

## 📈 Принцип работы и архитектура

Бот спроектирован с учетом жестких ограничений встраиваемых систем:
1. **Чистый Shell**: Логика бота не использует Python, Node.js или Lua-скрипты. Бот написан на чистом `/bin/sh`.
2. **Низкоуровневые утилиты**: Для разбора JSON используется встроенный в OpenWRT быстрый бинарный парсер `jsonfilter`.
3. **Обмен данными**: Сетевые запросы осуществляются через `curl` с жестко лимитированными таймаутами (Long Polling), что исключает зависание процессов.
4. **Сохранение ресурса Flash-памяти**: Бот развертывает все временные файлы, кэш обновлений и логи исключительно в каталоге `/tmp` (оперативная память `tmpfs`), полностью оберегая физическую Flash-память роутера от износа записи.
5. **Встроенный демон**: Служба оформлена как стандартный системный сервис `init.d` с автоматическим запуском при загрузке роутера.

---

## ⚡ Потребление ОЗУ и памяти

* **Оперативная память (RAM)**: **менее 2 МБ** во время работы (в пиках Long Polling). В режиме ожидания нагрузка практически равна нулю, так как исполняется только стандартный Shell-интерпретатор.
* **Постоянная память (Flash)**: **менее 60 КБ** для всей установки (включая все скрипты, файлы локализации, LuCI-контроллеры и шаблоны).
* **Нагрузка на CPU**: Менее **0.1%** на стандартных роутерах (вычисления происходят только в момент обработки команд от пользователя).

## ✨ Основные возможности

* **📊 Мониторинг системы**: Просмотр Uptime, версий ПО, свободной памяти ОЗУ/Flash, загрузки процессора CPU, WAN IP-адресов.
* **⚡ Управление сетевыми интерфейсами**: Включение/выключение любого интерфейса прямо в чате с кнопками подтверждения действия.
* **📱 Список подключенных устройств**: Мгновенный вывод активных DHCP-аренд с именами устройств, IP и MAC-адресами.
* **🔄 Перезагрузка**: Безопасная инициация перезагрузки роутера удаленно с клавиатуры бота.
* **🌐 Мультиязычность (Локализация)**: Поддержка 4 языков (Русский 🇷🇺, English 🇬🇧, Українська 🇺🇦, Deutsch 🇩🇪), которые переключаются «на лету» в боте или через LuCI.
* **🔔 Уведомления**: 
  * При подключении новых устройств к сети (по MAC-адресу).
  * При изменении внешнего WAN IP-адреса.
  * При критической загрузке процессора (CPU) или нехватке ОЗУ (настраиваемые пороги в %).
  * При выходе обновлений бота.
* **🔐 Безопасность и Авторизация**: Строгий доступ только для администраторов из белого списка (задается через LuCI). Попытки доступа посторонних лиц блокируются и логируются.
* **📁 Управление логами в LuCI**: Интегрированное модальное окно просмотра логов активности и несанкционированного доступа с возможностью настройки авто-очистки по времени или количеству строк.

## 🛠 Быстрая установка

Для установки требуются права `root` и OpenWRT версии 24+ (с поддержкой пакетного менеджера `apk`).

1. Подключитесь к роутеру по SSH.
2. Скачайте архив проекта в `/tmp`:
   ```bash
   wget -O /tmp/bot.tar.gz https://github.com/vlam1n-dev/OpenWRT-TelegramBot/archive/refs/heads/main.tar.gz
   ```
3. Распакуйте архив и перейдите в папку:
   ```bash
   tar -xzf /tmp/bot.tar.gz -C /tmp/
   cd /tmp/OpenWRT-TelegramBot-main
   ```
4. Сделайте установщик исполняемым и запустите его:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```

После успешной установки перейдите в веб-интерфейс LuCI роутера: **Службы ➡️ Telegram Bot**, введите токен вашего бота (полученный у `@BotFather`), добавьте свой Telegram ID в список администраторов и включите службу.

---

## 🧹 Удаление проекта

Для полного и безопасного удаления бота и всех его компонентов (не затрагивая другие части LuCI) запустите деинсталлятор:
```bash
cd /tmp/OpenWRT-TelegramBot-main 2>/dev/null || cd /tmp/telegram-bot
chmod +x uninstall.sh
./uninstall.sh
```
