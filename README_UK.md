# 🤖 OpenWRT Telegram Bot

**OpenWRT Telegram Bot** — це ультра-легковажний, швидкий та безпечний Telegram-бот для віддаленого моніторингу та керування роутерами на базі OpenWRT. Написаний повністю на чистому командному інтерпретаторі POSIX Shell з використанням вбудованих API роутера (`ubus`, `jsonfilter`), що робить його ідеальним рішенням для вбудованих систем з жорсткими обмеженнями ресурсів.

Проект містить повноцінний веб-інтерфейс адміністратора для **LuCI**, що дозволяє налаштовувати токен, список адміністраторів, сповіщення та логи прямо через стандартний кабінет роутера в браузері.

---

## 📈 Принцип роботи та архітектура

Бот спроектований з урахуванням жорстких лімітів вбудованих систем:
1. **Чистий Shell**: Логіка бота не використовує важкі залежності на кшталт Python, Node.js або Lua-скриптів. Бот написаний на чистому `/bin/sh`.
2. **Низькорівневі утиліти**: Для аналізу JSON використовується вбудований в OpenWRT швидкий бінарний парсер `jsonfilter`.
3. **Обмін даними**: Мережеві запити здійснюються через `curl` з жорстко лімітованими таймаутами (Long Polling), що запобігає зависанню процесів.
4. **Збереження ресурсу Flash-пам'яті**: Бот розгортає всі тимчасові файли, кеш оновлень та логи виключно в каталозі `/tmp` (оперативна пам'ять `tmpfs`), повністю оберігаючи фізичну Flash-пам'ять роутера від зносу запису.
5. **Вбудований демон**: Служба оформлена як стандартний системний сервіс `init.d` з автоматичним запуском при завантаженні роутера.

---

## ⚡ Споживання ОЗУ та пам'яті

* **Оперативна пам'ять (RAM)**: **менше 2 МБ** під час роботи (у піках Long Polling). В режимі очікування навантаження на процесор та ОЗУ практично дорівнює нулю.
* **Постійна пам'ять (Flash)**: **менше 60 КБ** для всієї інсталяції (включаючи всі скрипти, файли локалізації, LuCI-контролери та шаблони).
* **Навантаження на CPU**: Менше **0.1%** на стандартних роутерах (обчислення відбуваються тільки в момент обробки команд від користувача).

## ✨ Основні можливості

* **📊 Моніторинг системи**: Перегляд Uptime, версій ПЗ, вільної пам'яті ОЗУ/Flash, завантаження процесора CPU, WAN IP-адрес.
* **⚡ Керування мережевими інтерфейсами**: Увімкнення/вимкнення будь-якого інтерфейсу прямо у чаті з кнопками підтвердження дії.
* **📱 Список підключених пристроїв**: Миттєве виведення активних оренд DHCP з іменами пристроїв, IP та MAC-адресами.
* **🔄 Перезавантаження**: Безпечна ініціація перезавантаження роутера віддалено з клавіатури бота.
* **🌐 Мультиязичність (Локалізація)**: Підтримка 4 мов (Українська 🇺🇦, Русский 🇷🇺, English 🇬🇧, Deutsch 🇩🇪), які перемикаються «на льоту» в боті або через LuCI.
* **🔔 Розумні сповіщення**:
  * При підключенні нових пристроїв до мережі (за MAC-адресою).
  * При зміні зовнішньої WAN IP-адреси.
  * При критичному завантаженні процесора (CPU) або нестачі ОЗУ (налаштовувані пороги у %).
  * При виході оновлень бота.
* **🔐 Безпека та авторизація**: Доступ суворо обмежений адміністраторами з білого списку UCI. Спроби взаємодії сторонніх осіб блокуються та логуються.
* **📁 Керування логами в LuCI**: Інтегроване модальне вікно перегляду логів активності та несанкціонованого доступу з можливістю налаштування авто-очищення за часом або кількістю рядків.

## 🛠 Швидка інсталяція

Потрібні права `root` та OpenWRT версії 24+ (з підтримкою пакетного менеджера `apk`).

1. Підключіться до роутера по SSH.
2. Завантажте архів проекту в `/tmp`:
   ```bash
   wget -O /tmp/bot.tar.gz https://github.com/vlam1n-dev/OpenWRT-TelegramBot/archive/refs/heads/main.tar.gz
   ```
3. Розпакуйте архів та перейдіть до теки:
   ```bash
   tar -xzf /tmp/bot.tar.gz -C /tmp/
   cd /tmp/OpenWRT-TelegramBot-main
   ```
4. Запустіть скрипт інсталяції:
   ```bash
   chmod +x install.sh
   ./install.sh
   ```
5. Видаліть тимчасові інсталяційні файли з `/tmp`, щоб звільнити оперативну пам'ять роутера:
   ```bash
   rm -rf /tmp/bot.tar.gz /tmp/OpenWRT-TelegramBot-main
   ```

Після успішної інсталяції перейдіть у веб-інтерфейс LuCI роутера: **Служби ➡️ Telegram Bot**, введіть токен вашого бота (отриманий у `@BotFather`), додайте свій Telegram ID до списку адміністраторів та увімкніть службу.

## 🧹 Видалення проекту

Для повного та безпечного видалення бота та всіх його компонентів без зачіпання інших частин LuCI:
```bash
cd /tmp/OpenWRT-TelegramBot-main 2>/dev/null || cd /tmp/telegram-bot
chmod +x uninstall.sh
./uninstall.sh
```
