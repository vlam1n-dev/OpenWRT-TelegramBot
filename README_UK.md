# Що це?

**OpenWRT Telegram Bot** — це легковажний, швидкий та безпечний Telegram Бот для віддаленого моніторингу та управління роутерами на базі OpenWRT. Написаний повністю на чистому Shell з використанням вбудованого API роутера (`ubus`, `uci`, `jsonfilter`) та API Telegram.

Налаштування конфігураційного файлу доступне як через конфіг-файл `/etc/config/telegram`, так і через веб-інтерфейс **LuCI** - налаштування токена, списку адміністраторів, сповіщень у боті, перевірка оновлень та виведення логів `Services -> Telegram Bot`

## Архітектура

- **Shell**: Чистий `/bin/sh`;
- **Низькорівневі утиліти**: Для розбору JSON використовується вбудований бінарний парсер `jsonfilter`;
- **Обмін даними**: Мережеві запити здійснюються через `curl`;
- **Збереження ресурсу Flash-пам'яті**: Тимчасові файли розташовуються в каталозі `/tmp`;
- **Вбудований демон**: Бот працює як системна служба під управлінням супервізора `procd`.

---

## Системні вимоги

- **Операційна система**: OpenWRT **версії 19.07 і вище**;
- **Базові залежності** (встановлюються автоматично):
  - `curl`;
  - `jsonfilter`;
  - `ca-certificates` / `ca-bundle`.
- **Додаткові залежності** (встановлюються за бажанням в `install.sh` для роботи розширених модулів):
  - `etherwake` або `wol` - для функції Wake-on-LAN;
  - `vnstat` / `vnstatd` - для збору статистики загального трафіку;
  - `nlbwmon` - для обліку споживання мережі локальними клієнтами.

## Основні можливості

- **📊 Моніторинг системи**:
  - Перегляд **аптайму**, **версії ПЗ**, **вільної пам'яті** ОЗП/Flash, **завантаження процесора** CPU, **WAN IP**-адрес;
  - **Запуск** стандартних **мережевих утиліт** з динамічним виведенням результатів у реальному часі (Ping, Traceroute, DNS Lookup, Port Check).
- **⚡ Управління мережевими інтерфейсами**:
  - **Увімкнення/вимкнення** будь-якого **мережевого інтерфейсу**, доступного в OpenWRT;
  - Перегляд детальної інформації про кожен мережевий інтерфейс.
- **📱 Список підключених пристроїв**
  - Миттєве **виведення активних DHCP-оренд** з детальною інформацією, що включає IP, інтерфейс підключення та багато іншого;
  - Можливість відключити від мережі будь-який пристрій;
  - Можливість заблокувати пристрій для подальшого підключення до даного мережевого інтерфейсу;
  - Можливість надіслати **Wake-on-Lan** пакет.
- **🔀 Прокидання портів (NAT)**
  - **Управління файрволом** роутера без входу в LuCI;
  - Створення, зміна, увімкнення/вимкнення вже існуючих правил.
- **🔄 Перезавантаження**
  - Перезавантаження будь-якого мережевого інтерфейсу, включаючи сам роутер.
- **🔔 Сповіщення**
  - **Налаштування сповіщень** через Telegram бот (Підключення пристрою / Зміна WAN IP / Високе навантаження на систему).
- **🌐 Локалізація**
  - **Підтримка 4-х мов**, включає російську, англійську, українську, німецьку.
- **📁 Інтеграція з LuCI**
  - Налаштування API ключа, ID адміністраторів, сповіщень через бота, перевірка оновлень та багато іншого;
  - Перегляд детальних логів роботи Telegram Бота.

## Про споживання

* **Оперативна пам'ять**: **менше 2 МБ** під час роботи. У режимі очікування навантаження практично дорівнює нулю;
* **Постійна пам'ять (Flash)**: **менше 60 КБ** для всієї установки;
  - Усі змінні файли (база даних станів користувачів, тимчасові FIFO-канали, PID-файли, логи) винесені в директорію `/tmp`.
* **Навантаження на CPU**: Менше **1%**.

## Встановлення

Для встановлення потрібна OpenWRT версії `19.07` і вище.

**УВАГА**: Тестування проводилося тільки мною на версії `25.12.0`, на інших версіях можуть бути помилки!

1. Підключіться до роутера (наприклад, використовуючи [Putty](https://github.com/putty-org-ru/PuTTY/releases));
2. Запустіть **команду для автоматичного завантаження** та запуску встановлювача:
   ```bash
   wget -qO- https://raw.githubusercontent.com/vlam1n-dev/OpenWRT-TelegramBot/main/install.sh | sh
   ```
3. Пройдіть встановлювач, дотримуючись вказівок.

Після успішного встановлення перейдіть у веб-інтерфейс LuCI роутера (якщо був встановлений): `Services -> Telegram Bot`, введіть токен вашого бота (отриманий у [@BotFather](https://t.me/BotFather)) та додайте свій **Telegram ID** до списку адміністраторів.

Оновлення відбувається точно так само, запустивши `install.sh` вкажіть **нову версію** або встановіть **автоматично**.

## Видалення

Для повного та безпечного видалення бота та всіх його компонентів, запустіть деінсталятор:
```bash
wget -qO- https://raw.githubusercontent.com/vlam1n-dev/OpenWRT-TelegramBot/main/uninstall.sh | sh
```

Ви можете вибрати, які **компоненти видалити**, а які **залишити**, у тому числі - **конфігураційний файл**.
