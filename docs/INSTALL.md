# 📘 Полное руководство по установке и настройке OpenWRT Telegram Bot

В этом документе пошагово описан весь процесс: от подготовки Telegram-бота и настройки репозитория до окончательной установки на роутер OpenWRT.

---

## Этап 1: Подготовка в Telegram

Перед установкой на роутер вам необходимо получить токен бота и ваш личный идентификатор (ID).

1. **Создание бота:**
   - Откройте Telegram и найдите официального бота **[@BotFather](https://t.me/BotFather)**.
   - Напишите команду `/newbot`.
   - Придумайте имя вашему боту (например, `Мой OpenWRT Роутер`).
   - Придумайте уникальный username (обязательно должен заканчиваться на `bot`, например `vlam1n_openwrt_bot`).
   - В ответ BotFather пришлет вам **API Token** (выглядит примерно так: `1234567890:ABCdefGhIJKlmNoPQRstuVWXyz`). Обязательно скопируйте его, он понадобится позже.

2. **Получение вашего ID администратора:**
   - Найдите в Telegram бота **[@getmyid_bot](https://t.me/getmyid_bot)** (или любого аналогичного).
   - Нажмите "Запустить" (Start).
   - Бот ответит сообщением. Скопируйте цифры из поля `Your user ID:` (например, `123456789`). Это нужно, чтобы никто, кроме вас, не мог управлять роутером.

## Этап 2: Подготовка GitHub (Создание релиза)

Ваш бот умеет автоматически проверять наличие обновлений на GitHub. Для того чтобы это работало корректно, нам нужно создать "Релиз" (Release). Вы делаете это один раз для текущей версии, а затем повторяете при выходе новых.

1. Откройте ваш репозиторий на сайте: `https://github.com/vlam1n-dev/OpenWRT-TelegramBot`
2. Справа, в блоке "Releases", нажмите на **Create a new release** (или перейдите по ссылке `https://github.com/vlam1n-dev/OpenWRT-TelegramBot/releases/new`).
3. В поле **Choose a tag** введите `v1.0.0` и нажмите *Create new tag: v1.0.0 on publish*.
4. В поле **Release title** напишите `Версия 1.0.0 — Первый релиз`.
5. В большое текстовое поле можно добавить описание: `Базовая версия бота с поддержкой управления интерфейсами и статистикой`.
6. Нажмите зеленую кнопку **Publish release**.

Отлично! Теперь система обновлений бота будет знать актуальную версию проекта.

## Этап 3: Установка на OpenWRT

Теперь всё готово для загрузки скриптов непосредственно на роутер. 

> **Требования:** Роутер на базе OpenWRT 24.x (с менеджером пакетов `apk`) и доступ в интернет.

1. Откройте терминал на вашем компьютере (PowerShell, PuTTY или терминал macOS/Linux).
2. Подключитесь к роутеру по SSH (замените IP на свой):
   ```bash
   ssh root@192.168.1.1
   ```
3. Выполните следующие команды по очереди, чтобы скачать и запустить установщик:

   ```bash
   # 1. Переходим во временную память (чтобы не расходовать ресурс флеш-памяти)
   cd /tmp

   # 2. Скачиваем актуальный архив с вашего GitHub
   wget -O bot.tar.gz https://github.com/vlam1n-dev/OpenWRT-TelegramBot/archive/refs/heads/main.tar.gz

   # 3. Распаковываем архив
   tar -xzf bot.tar.gz

   # 4. Переходим в папку с файлами
   cd OpenWRT-TelegramBot-main

   # 5. Даём установщику права на запуск
   chmod +x install.sh

   # 6. Запускаем автоматическую установку
   ./install.sh
   ```

Скрипт сам скачает недостающие системные утилиты (curl, пакеты для LuCI), распределит файлы по нужным папкам и запустит сервис.

4. (Опционально) Очистите оперативную память от скачанных архивов:
   ```bash
   rm -rf /tmp/bot.tar.gz /tmp/OpenWRT-TelegramBot-main
   ```

## Этап 4: Настройка через веб-интерфейс LuCI

Файлы установлены, теперь нужно передать боту ваши токен и ID.

1. Откройте веб-интерфейс роутера в браузере (например, `http://192.168.1.1`).
2. В верхнем меню перейдите в раздел **Services** (Службы) -> **Telegram Bot**.
3. **General Settings (Основные настройки):**
   - Установите галочку **Enable Service** (Включить сервис).
   - Вставьте **API Token**, который дал BotFather.
   - Измените **Bot Language** на *Русский* (если нужно).
4. **Administrators (Администраторы):**
   - Нажмите `Add` (Добавить) и введите ваш цифровой Telegram ID.
5. **Notifications (Уведомления):**
   - Выберите, о чем бот должен присылать уведомления (новые устройства, высокая нагрузка, смена IP, наличие новых релизов на GitHub).
6. В самом низу нажмите **Save & Apply** (Сохранить и применить).
7. Вернитесь наверх страницы и нажмите кнопку **Test API**. Если всё введено верно, вы увидите зеленое сообщение "*Success! Bot name: @ваш_бот*".

## Завершение

Поздравляем! Откройте диалог с вашим ботом в Telegram и напишите команду `/start`.
Появится меню с полным отчетом о системе и кнопками управления.

### Как удалить бота?
Если вы решите удалить бота с роутера, просто подключитесь по SSH и выполните:
```bash
/usr/bin/telegram-bot-uninstall.sh # Если вы перенесли файл деинсталлятора
# Или скачайте деинсталлятор отдельно:
cd /tmp
wget https://raw.githubusercontent.com/vlam1n-dev/OpenWRT-TelegramBot/main/uninstall.sh
chmod +x uninstall.sh
./uninstall.sh
```
